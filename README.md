# Tigris & Euphrates — Online

Онлайн-версия настольной игры Tigris & Euphrates (Тигр и Евфрат) для 2–4 игроков.

## Концепция
Цифровой стол без движка: игроки сами принимают игровые решения, общаясь по видеозвонку.  
Автоматизируется только раздача тайлов из мешка.

## Как играть
1. Открыть сайт
2. Выбрать количество игроков (2, 3 или 4)
3. Нажать «Начать игру»
4. Выбрать тайл или лидера в нижней панели
5. Кликнуть на клетку поля для размещения
6. Завершить ход — тайлы автоматически доберутся из мешка

## Механика
- 2 действия за ход: разместить тайл, поставить/убрать лидера, обменять тайлы
- Конфликты разрешаются игроками вручную (убрать тайлы кнопкой «Убрать»)
- Очерёдность, мешок и раздача — автоматически

## Стек
- Чистый HTML + CSS + JS (без фреймворков)
- Хостинг: Netlify

## Структура
```
te-game/
├── index.html          # Вся игра
└── assets/
    ├── board/field.png # Фон поля
    ├── tiles/          # 6 тайлов

    └── cubes/          # Кубики очков
```
